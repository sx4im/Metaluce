# Metaluce

**Transform Meeting Chaos into Clear Action**

## Overview
Metaluce is an AI-powered meeting transcript analyzer that transforms chaotic meeting notes into clear, actionable insights. The name combines "Meta" (beyond) with "Luce" (light/clarity in Italian), representing our mission to bring clarity to meetings.

## Features
- **Transcript Input**: Paste text or upload .txt files
- **AI Analysis**: Intelligent summarization and action item extraction
- **Executive Dashboard**:
  - Executive Summary
  - Prioritized Action Items
  - Kanban Board (High/Medium/Low priority)
- **Beautiful UI**: Modern glassmorphism design with 3D interactions

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn UI, Framer Motion
- **Backend**: Express, Drizzle ORM, PostgreSQL
- **3D Graphics**: Spline (@splinetool/react-spline)
- **AI**: Mock implementation for development

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Project Structure
- `/client` - React frontend application
- `/server` - Express backend API
- `/shared` - Shared types and utilities
